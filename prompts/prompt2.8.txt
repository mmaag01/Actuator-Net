# Context

Test 5b (if it verdicts "torEst carries state" or "Ambiguous") showed that the 30-sample window is too short to resolve spring-ringing events. The loading phase that produced the trapped torque is more than 30 ms back. A longer window lets the GRU see the loading phase and learn "motor was recently commanded +140 Nm followed by sudden release → expect slow decay."

Run this ONLY after Prompt 7 completes and reports its verdict.
