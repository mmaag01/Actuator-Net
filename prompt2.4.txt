# Context

The architecture-invariance result from Test 4 (corr(pred_mlp, pred_gru) = 0.999) combined with Test 1's finding that torEst-alone is RMSE 19.8 Nm suggests the models converged on a conditional mean driven by some subset of features. To understand what they're actually using, ablate torDes and i one at a time and measure the impact on per-regime performance.

This should be run AFTER prompt 3b (use the v2 models and data), so the results reflect the hold-enriched dataset.

# Task

# Interpretation block

After the table, include a written interpretation (Claude Code should fill this in based on the numbers):

- "torDes is load-bearing" if Config B degrades hold RMSE by >30% vs A
- "i is load-bearing" if Config C degrades osc or transition RMSE by >30% vs A
